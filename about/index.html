About Me

Posted at — May 23, 2020

Hi! I’m Steven, a software engineer living in the Philippines. I mostly write software in Go but I play around with other programming languages just for the fun of it.

I prefer Vue.js over React and my go to css framework is Bulma but I usually use Buefy coz it makes it so much easier to use Vue.js and Bulma together.

Currently, I’m working on my personal projects and in the process, came across some interesting technologies that I’ve never used before such as Solr. It’s very similar to Elasticsearch. In fact, they use the same underlying full-text search library called Lucene. Learning Solr has been an amazing experience so far.
